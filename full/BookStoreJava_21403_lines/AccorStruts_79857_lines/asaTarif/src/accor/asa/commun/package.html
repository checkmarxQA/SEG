Ce package contient toutes les classes potentiellement partageables avec d'autres projets Accor. A l'heure actuelle, ce package ne contient que les classes relatives à l'internationalisation (sous-package com.accor.commun.internationalisation). Les classes présentes à ce niveau ont en partie été récupérées du projet Accor-Hotels.com, et permettent d'accéder simplement aux différents labels traduits de l'application. Remarque 1 : le projet Accor-Hotels.com n'utilise pas exactement la même version de ces classes (elles ont été déplacées dans un package différent, commun, en vue d'une unification de code des deux applications, mais Accor-Hotels n'a pas encore, à notre connaissance, intégré ces modifications) Remarque 2 : les classes livrées par Accor-Hotels ont été étendues pour permettre au projet ASA d'aller récupérer les libellés des différents champs depuis une base de données.
